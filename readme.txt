Для работы этого шаблана необходимо:
1. npm i - инициализация нового проекта
2. Выделить основные стили для конкретного проекта
3. Выбрать способ адаптивной верстки (брейк-поинты или отзывчивая)

//============================================================

Команды gulp сборки:

npm run svgSprive - создание svg спрайта
npm run dev - запуск gulp в режиме разработчика (когда разрабатывается сайт)
npm run build - запуск gulp в режиме продакшена (формируются конечные файлы перед отправкой заказчику)
npm run zip - создание архива с результатом

npm i - инициализация нового проекта

gulp --v - Версия gulp

//============================================================
